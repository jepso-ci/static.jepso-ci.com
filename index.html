JEPSO-CI Static

Welcome to JEPSO-CI's static server. It hosts JavaScript files and Stylesheets. That may be useful when constructing your tests.

It's perfectly fine to directly reference the scripts on here, the advantage of which is that you get to always have the latest versions of all of them. Alternatively, you can download them locally so they work even offline.

The website is also fully https so you can use it without worrying about running into annoying cerficiate issues.

The files are not minified, and will not be minified as it simplifies debugging.

Test Harnesses & Adapters

Mocha

Mocha is an extensive and powerful test framework built for node.js and the browser. You can see documentation for it here. To use it with jepso-ci you'll need the following three files.

  • https://static.jepso-ci.com/adapters/mocha.js
  • https://static.jepso-ci.com/harnesses/mocha/mocha.js
  • https://static.jepso-ci.com/harnesses/mocha/mocha.css

To get started, you might want to try this as your html template:

<!doctype html>
<html>
<head>
  <meta charset="utf-8">
  <title>Mocha Tests</title>
  <link rel="stylesheet" href="https://static.jepso-ci.com/harnesses/mocha/mocha.css" />
</head>
<body>
  <div id="mocha"></div>
  <script src="https://static.jepso-ci.com/harnesses/mocha/mocha.js"></script>
  <script src="https://static.jepso-ci.com/adapters/mocha.js"></script>
  <script src="tests.js"></script>
  <script>mocha.run();</script>
</body>
</html>
